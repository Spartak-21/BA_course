# Weather Data Business Intelligence Task

## Overview

This project demonstrates how to collect real-time weather data from the [Weatherstack API](https://weatherstack.com/) using Python, and save it to a CSV file for business intelligence analysis. The script can be easily adapted to collect data for multiple cities.

## Features

- Fetches current weather data for one or more cities.
- Extracts key metrics: temperature, humidity, weather description, wind speed.
- Saves results to a CSV file for further analysis in Excel, Power BI, or other BI tools.
- Easily extendable to other APIs or more cities.

## Requirements

- Python 3.x
- `requests` library (install with `pip install requests`)
