# Power BI Sales Performance Dashboard

This folder contains the Power BI dashboard and related files for the Classic Models sales performance analysis.

## Files

- **PowerBI_dashboard.pbix**  
  The original Power BI Desktop file. Open with Power BI Desktop to explore or edit the report.

- **PowerBI_dashboard.pdf**  
  A PDF export of the dashboard for quick viewing without Power BI Desktop.

## Dashboard Overview

The dashboard provides:
- Key KPIs: Total Sales, Number of Orders, Number of Customers
- Sales trends by month
- Top products and customers by sales
- Sales by country (map visualization)
- Interactive slicers for date, product line, and order status

## How to Use

1. Open `PowerBI_dashboard.pbix` in Power BI Desktop for full interactivity.
2. View `PowerBI_dashboard.pdf` for a static overview.
